# OpenSnap

OpenSnap is a free alternative to Razer's Snap Tap, designed to improve your counter-strafing techniques in games like Counter-Strike and Valorant. This application allows you to customize keybindings and automate movement controls for precise and efficient gameplay.

## Features

- **Simple Interface:** Easy-to-use GUI for quick setup.
- **Customizable Keybindings:** Adjust keys for your preferred gameplay style.
- **Safe and Non-Intrusive:** Uses Windows API for key detection without injecting into the game.
- **Autostop Feature:** Automatically stops player movement for improved accuracy.

## Installation

1. Download the latest release from [OpenSnap.tech](https://opensnap.tech).
2. Extract the ZIP file.
3. Run `OpenSnap.exe`.

## Usage

1. Open the application.
2. Customize your keybindings in the interface.
3. Toggle the autostop feature if needed.
4. Press "Start" to enable the hook and start using OpenSnap in your game.

## Keybinding Configuration

You can customize the keys for movement (W, A, S, D) by entering the desired keys in the provided text fields and clicking "Update Settings".

## Autostop Configuration

The autostop feature allows you to set the duration (in milliseconds) for which the opposite strafe key will be held down, ensuring the player comes to a full stop. You can enable this feature through the checkbox and set the desired delay in the interface.
